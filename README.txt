Created a new plugin to add a new icon and while clicking on that icon get a list of available links from 
database in according.
Click on any available links it will be added on selected text into the CKEditor.

Step 1. Copy and paste the 'trackinglink' folder into plugin folder
Step 2. Open the config.js file from CKEditor.
Step 3. Add "config.extraPlugins = 'trackinglink';".
Step 4. Add "TrackingLink" text on the tollbar options.
Step 5. For Step 3 and 4 get help from the config.js file attached into the folder.

Just refresh the page and enjoy you project. :)
